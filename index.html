  1. 追踪要控制版本的文件 git add 文件名(将文件提交到暂存区域)
  2. 提交该版本内容 git commit -m "版本描述"
通过 git status 查看文件的状态
  1. 已修改
  2. 已暂存
  3. 已提交
移除文件 如果是直接从系统中删除文件，注意一定要使用 git rm 文件名的方式，将该文件从暂存区域中删除 1. rm 文件名 （从系统中删除文件） 2.git rm 文件名(从暂存区域中删除文件) 创建分支
  1. git branch testing 创建分支
  2. git checkout testing(master) 切换分支
合并分支
  1. 创建分支和切换分支 git checkout -b 分支名称
  2. 完成代码编写
  3. 切换回主分支
  4. 将子分支合并到主分支中 git merge 自分支名称
